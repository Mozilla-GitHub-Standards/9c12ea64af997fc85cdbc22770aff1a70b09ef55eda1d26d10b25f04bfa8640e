Firefox a WebExtensions

  • 1/3 doplňků stačí „přebalit“
  • nadstavba nad API Chrome
  • W3C standard
  • ne všichni implementují všechno
XUL používá jen Mozilla
XUL dává doplňkům přístup prakticky k čemukoliv
Firefox (zatím) neimplementuje vše, co Chrome (deprecated nebo málo používané)
Firefox slibuje přidat nadstavby, o které mají vývojáři zájem
WebExtensions možná standardem
